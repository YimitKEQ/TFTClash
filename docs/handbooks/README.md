# TFT Clash Handbooks

This directory holds the official handbooks for TFT Clash. They're written to be readable on their own (open the `.md` file in any markdown viewer or render on GitHub) and the SVG diagrams are embedded inline so everything looks right both online and printed.

Use them to onboard new hosts, explain the platform to sponsors, or hand to a player who's never registered before.

| Handbook | For | One-line pitch |
|----------|-----|----------------|
| [HOST_HANDBOOK.md](./HOST_HANDBOOK.md) | Tournament organizers | Run branded events, keep 70-100% of revenue, $19.99/mo flat. |
| [PLAYER_HANDBOOK.md](./PLAYER_HANDBOOK.md) | Competitors | Free to compete. Climb the season ladder. Build a public resume. |
| [PLATFORM_OVERVIEW.md](./PLATFORM_OVERVIEW.md) | Sponsors, partners, anyone curious | What this is, why it exists, what makes it different. |
| [OPERATOR_GUIDE.md](./OPERATOR_GUIDE.md) | Internal / admin | How to run the platform itself - approvals, payouts, support. |

## Visual assets

All diagrams live in [`assets/`](./assets/) as SVGs. They're hand-tuned for the dark TFT Clash theme and render cleanly at any size.

- `platform-overview.svg` - the three-sided platform (players / hosts / spectators)
- `tournament-flow.svg` - 5-phase tournament lifecycle
- `host-revenue.svg` - the three revenue streams hosts can tap
- `scoring-table.svg` - official EMEA point system + tiebreakers

You can drop these into Notion, Google Slides, a pitch deck, or print them. They're CC-BY licensed for use by hosts and partners.

## Updating

These handbooks track product reality. When a feature ships or the pricing changes, edit the relevant handbook in the same PR. Don't let them drift.
